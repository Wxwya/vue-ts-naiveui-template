# 项目介绍
> 基于 vite + Vue3 + ts + pinia + tailwindcss 搭建的极简模板(自用)


- 项目图标库使用 iconify-json
- 封装了常用的 form, table, tabs 组件
- 支持布局更改，支持主题更改
- 项目预览地址：[在线地址](https://wxwya.github.io/naive_admin_template/)
